# Ad Code Snippets

Use the following code snippets to re-add ads to any page.

---

## 1. Head Script

Place this inside `<head>`, before `</head>`:

```html
<script src="https://aloudalimonyexplosion.com/ca/8b/51/ca8b5147d5b46238726f3d8f790975c8.js"></script>
```

---

## 2. Bottom-of-Page Banner Ad

Place this before `</body>`:

```html
<script async="async" data-cfasync="false" src="https://aloudalimonyexplosion.com/96d03b03f19222088db0ab75075de359/invoke.js"></script>
<div id="container-96d03b03f19222088db0ab75075de359"></div>
```

---

## 3. Inline Ad (used on the Free product page)

Place this wherever you want an inline ad within the page content:
