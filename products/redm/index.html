Product Description

Overview

Scooby RedM is a comprehensive mod menu designed for RedM, the multiplayer modification framework for Red Dead Redemption 2. Built specifically for roleplay servers and custom game modes, our menu provides powerful tools, PVP features, and extensive customization options while maintaining stability and performance.

Combat Features

  • ESP / Wallhacks - See players, NPCs, horses, and objects through terrain with customizable colors and distance filters
  • Aimbot - Precision aim lock with customizable FOV, smoothing, bone selection, and visibility checks
  • Triggerbot - Automatically fires when your crosshair is on target with customizable delay
  • Silent Aim - Hit targets without visible aim movement for stealthy elimination
  • No Recoil - Eliminate weapon recoil for perfect accuracy
  • Rapid Fire - Increase your weapon's fire rate beyond normal limits

Player Features

  • God Mode - Become invincible to all damage sources
  • Infinite Stamina - Never run out of stamina for sprinting or horse riding
  • Super Jump - Jump to incredible heights
  • No Clip - Fly through walls and terrain freely
  • Speed Hack - Increase your movement speed dramatically
  • Invisible - Become invisible to other players
  • Teleportation - Instantly travel anywhere on the map or to waypoints

Horse & Vehicle Features

  • Horse Spawner - Spawn any horse breed with custom stats and bonding level
  • Horse God Mode - Make your horse invincible
  • Infinite Horse Stamina - Your horse never gets tired
  • Horse Speed Boost - Increase your horse's speed beyond normal limits
  • Wagon Spawner - Spawn wagons and carriages
  • Vehicle Mods - Modify vehicle properties and handling

Weapon Features

  • Weapon Spawner - Spawn any weapon in the game instantly
  • Infinite Ammo - Never run out of ammunition
  • Weapon Customization - Modify weapon damage, range, and accuracy
  • Quick Reload - Instantly reload your weapons
  • Give All Weapons - Get every weapon in the game with one click

Server & Advanced Features

  • Lua Executor - Execute custom Lua scripts on RedM servers
  • Resource Manager - View, start, stop, and restart server resources
  • Event Manager - Monitor and trigger server events for exploits
  • Dumper - Dump server-side data, resources, and scripts
  • Triggers - Custom trigger system for automated actions and exploits
  • Anti-Cheat Checker - Detect which server-side anti-cheat is running before joining

Player Trolling

  • Cage Player - Trap players in cages
  • Explode Player - Create explosions around other players
  • Ragdoll Player - Make other players fall and ragdoll
  • Attach Objects - Attach objects to other players
  • Spectate - View through other players' cameras
  • Clone Player - Create clones of other players

Visual & Misc Features

  • Time & Weather Control - Change time of day and weather conditions
  • Ped Spawner - Spawn any NPC character
  • Model Changer - Change your player model to any character
  • Outfit Editor - Customize your clothing and appearance
  • Name Spoofer - Change your displayed name
  • Server Info - View detailed server information

Protection & Security

  • HWID Spoofer - Spoof your hardware IDs to bypass bans
  • Ban Cleaner - Clean ban traces and cached data from RedM
  • Anti-Cheat Bypass - Built-in bypasses for common RedM anti-cheats
  • Event Blocker - Block malicious server events

Menu Features

  • Customizable UI - Adjust colors, position, and size of the menu
  • Config System - Save and load your favorite settings
  • Hotkeys - Bind features to custom keyboard shortcuts
  • Search Function - Quickly find features by name
  • Favorites - Mark frequently used features for quick access
