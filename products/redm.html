Product Description

Overview

Scooby RedM is currently in development. RedM is the multiplayer modification framework for Red Dead Redemption 2, similar to FiveM for GTA V. Our menu will provide powerful tools for RedM roleplay servers.

Planned Features

  • ESP / Wallhacks - See players and objects through terrain
  • Aimbot - Precision aim with customizable smoothing
  • Horse Spawner - Spawn any horse breed instantly
  • Weapon Modifications - Enhanced weapons and infinite ammo
  • Teleportation - Travel anywhere on the map
  • Lua Executor - Execute custom scripts
