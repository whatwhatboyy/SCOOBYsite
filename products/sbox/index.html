Product Description

Overview

Scooby S&box is currently in development. S&box is the successor to Garry's Mod, built on Valve's Source 2 engine. Our menu will provide powerful tools and enhancements for all S&box game modes with full support for the new engine.

Planned Features

  • ESP / Wallhacks - See players, NPCs, and objects through walls with customizable visuals
  • Aimbot - Precision aim assistance with bone selection, FOV control, and smoothing
  • Triggerbot - Automatically fires when crosshair is on a valid target
  • No Recoil - Eliminate weapon recoil for perfect accuracy
  • Speed Hack - Move faster than normal players
  • Fly Hack - Fly freely around the map
  • God Mode - Become invincible to all damage
  • No Clip - Pass through walls and solid objects
  • Player Chams - Highlight players with custom colors and materials
  • Radar Hack - Mini-map overlay showing all player positions
